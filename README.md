# ⚡ VoltShop — Интернет-магазин бытовой техники

Курсовой проект. Многостраничный сайт интернет-магазина.

## 📁 Структура проекта

```
voltshop/
├── index.html          # Главная страница
├── catalog.html        # Каталог товаров
├── product.html        # Карточка товара
├── cart.html           # Корзина
├── css/
│   ├── style.css       # Общие стили (шапка, подвал, компоненты, темы)
│   ├── home.css        # Стили главной страницы
│   ├── catalog.css     # Стили каталога
│   ├── product.css     # Стили карточки товара
│   └── cart.css        # Стили корзины
├── js/
│   └── main.js         # JavaScript: тема, навигация, таймер, корзина
├── data/
│   └── products.xml    # Данные каталога в XML
└── README.md
```

## 🛠 Технологии (по ТЗ)

| Пункт ТЗ | Требование | Реализация |
|---|---|---|
| 3.2 | XML-формат данных | `data/products.xml` |
| 3.3 | HTML5 + XML | Семантическая разметка, XML данные |
| 3.4 | CSS3 / SCSS | CSS-переменные, `[data-theme]`, медиазапросы |
| 3.5 | Семантика HTML5 | `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`, `<figure>` |
| 3.5 | SVG | Логотип, иконки навигации, платёжные иконки в корзине |
| 3.5 | Несколько страниц | 4 страницы: Главная, Каталог, Товар, Корзина |
| 3.5 | JavaScript DOM | Тема, таймер, галерея, удаление из корзины, слайдер цены |
| 3.6 | Адаптивность | Breakpoints: 1024px, 768px, 480px |

## 🚀 Запуск

Откройте `index.html` в браузере. Никаких зависимостей не нужно.
